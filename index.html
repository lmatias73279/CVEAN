Estoy comprometido con la excelencia y preparado para aportar significativamente a cualquier organización.

Línea de Tiempo - Historia de HTML y CSS
1991
HTML 1.0

Primera versión de HTML creada por Tim Berners-Lee. Incluía elementos básicos para estructuración de documentos.

  • Etiquetas básicas para texto
  • Enlaces hipertexto
  • Listas simples
1995
HTML 2.0

Primera especificación oficial del W3C. Estableció estándares para formularios y tablas.

  • Formularios interactivos
  • Tablas para organización de datos
  • Mejora en la accesibilidad
1996
CSS Nivel 1

Primera especificación de Hojas de Estilo en Cascada. Separación de contenido y presentación.

  • Propiedades de fuente y color
  • Margenes y padding
  • Posicionamiento básico
1997
HTML 3.2

Incorporación de elementos para diseño más complejo y compatibilidad con navegadores.

  • Applets de Java
  • Scripts
  • Tablas más avanzadas
1998
CSS Nivel 2

Gran expansión de capacidades de diseño. Posicionamiento absoluto y relativo.

  • Posicionamiento avanzado
  • Modelo de caja completo
  • Medios de comunicación
1999
HTML 4.01

Última versión de HTML antes de XHTML. Mejora en accesibilidad y estructura.

  • Mejor separación estructura/presentación
  • Accesibilidad mejorada
  • Soporte para scripts y estilos
2014
HTML5

Revolución en la web con elementos semánticos y APIs para aplicaciones web complejas.

  • Elementos semánticos (header, nav, section, etc.)
  • APIs para gráficos y multimedia
  • Mejor soporte móvil
  • Almacenamiento local
